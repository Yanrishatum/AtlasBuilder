## Format

Every values is separated by new line.  
```
<total amount of frames>
[<total amount of frames>] <Delay of frame in seconds, Float value>
```
## Example
```
5    // Amount of frames
0.1  // Frame 0 delay
0.1  // Frame 1 delay
0.1  // Frame 2 delay
0.15 // Frame 3 delay
0.20 // Frame 4 delay
```
